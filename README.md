GreenTech Solutions Project
Welcome to my project. This is a multi-page website built to demonstrate the practical skills learnt in class. The site is designed for GreenTech Solutions, an eco-friendly consulting enterprise.
What This Project Does:
Carbon Footprint Calculator: Written in pure JavaScript. It grabs a user's weekly mileage and instantly calculates their yearly CO2 emissions directly on the screen without any page refreshes.
Consistent Design: Reuses CSS structures like .main-section and .box-container across pages so the user experience stays uniform.
No More Floating Footers: Solved that frustrating beginner bug where the footer floats in the middle of short pages by implementing smart minimum heights.
Built With:
HTML:Basic elements for clean, structured layouts.
CSS:Custom page grids,styling, and smooth hover states.
JavaScript: Detached event handling, calculations, and live text injection.
